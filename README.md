# Fitness ML Telegram Bot

Прототип системы анализа фитнес-данных с использованием машинного обучения.

## Функционал
- Интерактивный сбор данных через диалог в Telegram
- Автоматический расчёт ИМТ
- Классификация уровня подготовки (1–3) с помощью Random Forest
- Персонализированные рекомендации

## Требования
- Python 3.8+
- Библиотеки: `python-telegram-bot`, `pandas`, `scikit-learn`, `joblib`

## Как запустить
1. Установите зависимости: `pip install -r requirements.txt`
2. Обучите модель: `python train_model.py`
3. Получите токен у @BotFather, вставьте в `bot.py`
4. Запустите: `python bot.py`

## Структура проекта
- `src/` — модули с бизнес-логикой
- `train.py` — обучение модели
- `main.py` — запуск Telegram-бота
- `research.py` — проведение исследований
