# Fitness ML Bot

Telegram-бот для прогнозирования уровня физической подготовки на основе фитнес-данных с использованием модели машинного обучения.

## Функционал
- Интерактивный сбор данных через диалог в Telegram
- Автоматический расчёт ИМТ
- Классификация уровня подготовки (1–3) с помощью Random Forest
- Персонализированные рекомендации

## Требования
- Python 3.8+
- Библиотеки: `python-telegram-bot`, `pandas`, `scikit-learn`, `joblib`

Установка:
```bash
pip install -r requirements.txt
